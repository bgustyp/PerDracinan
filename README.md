# PerDracinan Streaming

Indonesian drama streaming website built with Next.js 16.

## Features

- Modern dark theme UI
- Responsive design
- Video player with auto-next
- Search functionality
- Watchlist management
- Continue watching

## Tech Stack

- Next.js 16.1.1
- TypeScript
- Tailwind CSS

## Installation

```bash
git clone https://github.com/bgustyp/PerDracinan.git
cd PerDracinan
npm install
```

Create `.env.local`:
```env
API_BASE_URL=
NEXT_PUBLIC_APP_NAME=PerDracinan
```

Run development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Usage

- Browse dramas on home page
- Search for specific content
- Add to watchlist
- Watch episodes with auto-advance
